TPWars

Interface Design

This personal project was born during Covid-19, with people stockpiling food and toilet paper which lead to an increasing waste and shortage of essential items. What if each person could know exactly how much of each item they need? And so, TPWARS was born.

Tools used: Adobe XD, illustrator, Visual Studio Code

Year: 2020

Programming Language: React

After testing a prototype with users there was a few design flaws that stand out:
- one of the things that stand out was it was not clear how users flipped a card. To solved this it was implemented a rounded corner on the cards.
This solution raised another problem, the location of the delete icon on the back of the card.

To solve the problem the consume add stock button was reduced, the consume button was moved to the front of the card, where it is easier for the user to consume an item, and the delete icon moved to the bottom back of the card. Below a screenshot taken from the prototype were the amends were done. This is a low fidelity prototype, so colours of the UI are not being taken in consideration.

All the buttons on the interface have a minimum clicking area of 48 by 48dp to make sure there's enough space for the users to click.
